## Install

```bash
npm i string-overlap-one-on-another
```

## Quick Take

```js
import { strict as assert } from "assert";
import overlap from "string-overlap-one-on-another";

assert.equal(overlap("aaa", "bbb", { offset: -2 }), "bbbaa");
```
